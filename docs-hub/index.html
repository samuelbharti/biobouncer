R and Python

biobouncer

Validate biological identifiers and inputs the same way in R and Python.

One small API answers "is this a valid identifier?" for gene symbols, ontology terms, variant formats, and database accessions. R and Python return the same verdict for the same input, checked against a shared conformance corpus. Choose an offline shape check, an offline existence check against a pinned snapshot, or a live check against the source.
